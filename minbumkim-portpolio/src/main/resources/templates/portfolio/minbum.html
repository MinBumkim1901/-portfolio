기술소개

2018 ~ 2022 대구과학대학교 컴퓨터정보통신과 프로그래밍에 대한 기초를 배우고 학습하였습니다
2021 ~ 2022 자격증 SQLD 데이터베이스의 기초를 배우고 습득하였습니다.
Front-End HTML

웹 페이지를 구성하는 기본 뼈대를 만들수 있으며 여러 가지 요소와 태그를 이용하여 웹 페이지의 레이아웃과 콘텐츠를 구성할 수 있습니다.
CSS

HTML로 구조를 생성한 후, CSS를 사용하여 해당 구조에 디자인과 스타일을 적용하여 웹 페이지를 아름답고 사용자 친화적으로 만들 수 있습니다.
JavaScript

XHR을 사용하여 서버로 HTTP 요청을 보내고, 서버로 요청을 처리해 데이터를 응답으로 반환해 비동기적인 통신 방식으로 웹 페이지와 통신할수 있습니다
Back-End Java

자바에서 제공하는 자료구조를 통해 데이터를 효율적으로 처리할수 있고 알고리즘을 통해 문제를 효율적으로 해결할수 있습니다.
Maria-DB

테이블을 만들어 삽입, 삭제, 수정이 가능하고 제약조건을 통해 불필요한 자료가 입력되는 것을 방지하기 위해 여러가지 규칙을 적용할수 있습니다.
SpringBoot

스프링 부트의 MVC를 통해 백엔드와 프론트 엔드 기반의 웹페이지를 개발하고 데이터베이스를 통해 CRUD를 할수 있습니다.
